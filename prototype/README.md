# Browser Motion Prototype

Open `index.html` in a browser or serve this folder with any static server.

Example:

```bash
python -m http.server 8000 -d prototype
```

Then open:

`http://localhost:8000`

## Included interactions

- scroll reveal and recede states,
- hero pointer parallax,
- CSS neural/data core,
- project-card 3D tilt,
- hover/focus/tap micro-demo playback,
- play/pause controls,
- scroll-driven depth scene,
- mobile layout,
- reduced-motion media-query support,
- manual motion toggle.

This prototype demonstrates the interaction language only. It is not the final résumé, and its copy and project facts must not be published without repository verification.
