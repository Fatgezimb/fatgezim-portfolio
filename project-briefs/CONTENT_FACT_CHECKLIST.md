# Content Fact Checklist

Use this checklist before publishing any copy.

## Identity and career

- [ ] Exact preferred display name
- [ ] Exact professional headline
- [ ] Current roles and organizations
- [ ] Exact legal/public organization names
- [ ] Employment and education dates
- [ ] Degree names
- [ ] Credential names and dates
- [ ] Contact links
- [ ] Résumé download source
- [ ] Location/availability wording

## Project facts

For every featured project:

- [ ] Exact public name
- [ ] Alias/previous name documented
- [ ] Owner's role verified
- [ ] Problem and audience verified
- [ ] Technology/method verified
- [ ] Status verified
- [ ] Outcome verified
- [ ] Public URL verified
- [ ] Screenshot/media permission verified
- [ ] Privacy boundary documented
- [ ] Micro-demo uses only local/sanitized data

## Known naming conflicts to resolve

- `NeuroPathLabs` vs `NeuroPath LLC`
- `BelaDataLabs` vs `Bela Data Lab` vs `Bela Data Lab Academy`
- `RBT-Pro Prep` vs `RBT Competency Prep` vs branded variants
- `Bela Caregiver Library` vs `Caregiver Academy`

## Prohibited claims without evidence

- users, customers, organizations, revenue, or growth
- percent improvements
- time saved
- accuracy or success rates
- clinical outcomes
- certification authority
- BACB affiliation or endorsement
- published research findings
- awards or rankings
- client identities

## Safety and privacy

- No PHI
- No client names
- No therapy session records
- No private credentials
- No secrets or internal URLs
- No protected training/test-bank content
