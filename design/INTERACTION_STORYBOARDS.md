# Interaction Storyboards

## Global project-card states

### Rest

- Project name, category, value statement, role, and status are readable.
- Static poster is visible.
- Buttons are present.
- No motion is required.

### Hover / focus

- Card raises 8–20 px visually.
- Border/elevation increases.
- Visual stage receives bounded 3D tilt.
- Text remains on a stable plane.
- After a short delay, the local micro-demo begins.
- Focus state is visible and not replaced by glow.

### Active / tap

- Preview controls become explicit.
- First tap activates preview; second explicit action opens details.
- On narrow screens, use a full-width preview sheet or inline expansion.

### Exit

- Card returns to rest smoothly.
- Demo pauses and resets or retains an understandable state.
- No audio autoplay.
