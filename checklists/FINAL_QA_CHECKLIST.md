- [ ] Install succeeds using the repository's package manager.
- [ ] Lint passes.
- [ ] Type check passes.
- [ ] Tests pass.
- [ ] Production build passes.
- [ ] No secrets are committed.

## Browser

- [ ] Desktop and mobile render correctly.
- [ ] Navigation works.
- [ ] All links work.
- [ ] Project previews work with mouse, keyboard, and touch.
- [ ] Reduced motion works.
- [ ] No console errors.
- [ ] No horizontal overflow.
- [ ] Print/download résumé works.

- [ ] Landmarks.

- [ ] Layout shifts controlled.
- [ ] Mobile interaction responsive.

- [ ] Base path correct.
- [ ] Routing correct.
- [ ] Assets load on deployed path.
- [ ] Workflow succeeded.
- [ ] Deployed page inspected.
