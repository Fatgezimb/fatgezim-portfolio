# Phase 2 Review Checklist

## Visual system

- [ ] Feels premium and distinctive, not generic.
- [ ] Still reads as a professional résumé.
- [ ] Contrast and typography are strong.
- [ ] Neon/glow are restrained.
- [ ] Brand names and assets are accurate.

## Project demos

- [ ] Every demo reflects a real workflow.
- [ ] Hover, focus, tap, and explicit controls work.
- [ ] Static/reduced-motion fallback works.
- [ ] No private data or fake metrics appear.
- [ ] Demo reset/pause behavior is understandable.

## Motion and 3D

- [ ] Scroll-in/out feels intentional.
- [ ] Pop-in/out is used selectively.
- [ ] 3D tilt remains readable.
- [ ] Major scenes are limited.
- [ ] No scroll hijacking.
- [ ] Mobile motion is appropriately reduced.
- [ ] Reduced-motion mode is complete.

## Performance and accessibility

- [ ] Keyboard navigation works.
- [ ] Focus is visible.
- [ ] 200% zoom works.
- [ ] No horizontal overflow.
- [ ] No console errors.
- [ ] Off-screen animation pauses.
- [ ] Static fallbacks load quickly.
