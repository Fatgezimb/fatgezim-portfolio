# Visual References

## Folders

### `generated-concepts/`

High-resolution original concept boards generated for this handoff.

These boards are visual-direction references only. They contain stylized placeholder copy and may visualize fictional metrics or states. Codex must replace all text, numbers, user names, progress values, and product claims with verified repository content.

### `markups/`

Editable SVGs and PNG exports:

1. Desktop content/layout approval wireframe
2. Project hover/focus/tap micro-demo storyboard
3. Scroll-in/out and 3D motion map
4. Mobile layout and reduced-motion behavior

### `renders/`

Focused concept crops for attaching to Codex one at a time.

## Asset rule

Do not directly ship placeholder metrics, fictional user names, imitation logos, or generated UI text from these references. Rebuild the visual language with verified content.
